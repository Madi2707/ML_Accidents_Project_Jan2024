This Application "Predicts Accidents Type" based on correlated conditions criteria.

-------------------------------------------------------------------------
In order to run the application use the following 3 steps from Terminal:
--------------------------------------------------------------------------
1)- Activate the virtual environment called venv using statement : .\venv\scripts\activate
2)- set the environment python path variable for this project directory using : $env:PYTHONPATH = "C:\Users\MAnsari\ml_accidents\"
3)- Then run the app on local port using command : python src\application\ml_app.py
